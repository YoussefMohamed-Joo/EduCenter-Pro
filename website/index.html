إدارة الطلاب

إضافة، تعديل، حذف، بحث، فلترة — مع باركود لكل طالب وبطاقة تعريفية قابلة للطباعة.

✅

الحضور والغياب

تسجيل الحضور بالباركود أو البحث السريع. إنشاء امتحانات تلقائياً لكل حصة.

📝

الامتحانات والدرجات

إنشاء امتحانات، إدخال درجات، عرض ترتيب الطلاب مع شارات ذهبية وفضية وبرونزية.

💰

المدفوعات والأقساط

تسجيل المدفوعات، نظام أقساط (3/4/6 دفعات)، إيصالات دفع مطبوعة مع QR.

📊

التقارير والتحليلات

تقارير الغائبين، المتبقين، غير المسددين، المتميزين — مع تصدير Excel, PDF, CSV.

👨‍🏫

المدرسون والموظفون

إدارة المدرسين، حضورهم، تقييماتهم. صلاحيات للموظفين (مشرف / عامل).

📱

الرسائل وواتساب

قوالب رسائل للواتساب، إشعارات غياب ودفع، فتح مباشر في واتساب ويب.

🎨

تخصيص كامل

تغيير الألوان، إضافة شعار المركز، تعديل اسم المركز — كل شيء قابل للتخصيص.

🔄

نسخ احتياطي

نسخ احتياطي واستعادة كاملين للبيانات. نسخ محلي وسحابي (Google Drive).
